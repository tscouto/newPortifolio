Sistema SGCCI

Sistema SGCCI

Ainda hoje em pleno século 21 vejo muita dificuldade para os profissionais na área de SST, realizar um verdadeiro controle de gestão de seus extintores e outros equipamentos de combate a incêndio. A maioria controla manualmente usando planilha simples ou texto e fica olhando no calendario quando que vai vencer é muito tempo perdido.

Lhe pergunto, imagina em uma empresa que tenha mais de 1000 extintores e 100 mangueiras de hidrantes. Como você vai controlar os vencimentos deles todos?

Nesse momento surgio a idéia de criar um verdadeiro sistema de gestão desses equipamentos. Onde tive o primeiro contato na programação na linguagem VBA. Onde me aprofundei na linguagem e com isso conseguir terminar o sistema.

Nesse projeto me surgiu novas ideias e no final criei um sitema chamado: SGCCI - Sistema de Gestão e Controle de Combate a Incêndio.

Como ele funciona?

O sistema ele tem os seguintes informações, como login e senha de acesso,cadastro de usuarios, cadastro de equipe de brigadistas,cadsatro de cipeiros, cadastro de extintores, cadastro de mangueira de hidrante, tela de acesso que filtra a lista dos equipamentos de extintores e de mangueiras de hidrantes que estão em dia, a vencer e vencido, além disso pode atualizar individual ou em cascata todos os equipamentos e também deletalos. Emissão de relátorio em pdf como: lista dos equipamentos ou relátorio anual com dashboard. Além de poder enviar automaticamente via e-mail o arquivo já anexado no e-mail com apenas um clique. E por último não menos importante o mesmo tem alerta automatico onde ele faz uma varredura automaticamente no banco de dados e verifica se tem algum equipamento que está para vencer. Caso tiver ele automaticamente avisa uma mensagem e mostra em tempo real, somente os equipamentos que estão para vencer. Resumindo é verdadeiro sistema fantástico para aqueles que trabalham na área de SST, e precisam de controle de automatização eficiente.

Na parte da programação foi usado linguagem backend VBA com banco de dados SQL para conectar o banco de dados ACCESS utilizando a biblioteca ADODB e o provedor de conexão Provider=Microsoft.ACE.OLEDB.12.0. Sendo que dentro do sistema VBA era conectado no banco de dados ACCESS, onde mandava e recebia os arquivos. Sem prejudicar o tamanho do arquivo original do excel.

Nesse projeto começou em setembro de 2022 até outubro de 2022.

Sistema de Controle de Atendimento Médico Lavori

Sistema Sistema de Controle de Atendimento Médico Lavori

Nessa época a empresa me solicitou um sistema para controle de gestão de atendimento médico. Como um bom soldado, missão dada é missão cumprida.

Nesse sistema conta com alguma informações uteis. Como cadastro, login e senha de usuário, médico, numero de atendimentos, agendamento de consultas, relatório em pdf, relátorio em dashboard.

Na linguagem de programação foi utilizado a linguagem VBA visual basic com comando SQL para conectar ao banco de dados ACCESS usando a biblioteca chamada ADODB.Usando também o provedor de conexão Provider=Microsoft.ACE.OLEDB.12.0.Onde mandava e recebia os arquivos, sem prejudicar o tamanho do arquivo original do excel.

Nesse projeto o tempo estimado foi de Junho de 2022 até Setembro de 2022.

Site job Finder

Site Job Finder

Nesse projeto foi utilizado linguagem como HTML, CSS, Javascript e principalmente a biblioteca Noje.js para conectar ao banco de dados SQLlite.

Como funciona o site?

Site é local de busca de vagas de empregos somente para ramo de programadores, onde tem a tela principal onde mostra a quantidade de vagas, outra tela você pode cadastrar a vaga. E ao clicar enviar ele automaticamente vai os dados para o banco de dados. Em seguida, na pagina oficial já adiciona a div.Na lista de vagas, aqui clicar na opção ver vaga, ele vai para outra pagina onde informa os detalhes da vaga. Como: Salários, nome da vaga, empresa e atividades que irá desenvolver.

Nesse projeto o tempo estimado foi de outubro de 2022 até dezembro de 2022.
